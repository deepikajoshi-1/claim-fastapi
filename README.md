# Claims Insurance Management System

A backend system built using FastAPI that manages insurance claims with secure authentication, role-based access control, and scalable architecture.

## Tech Stack
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- Role-Based Access Control (RBAC)
- Docker (planned)
- Kafka (planned)
- Azure (planned)

## Features
- User Registration & Login
- JWT-based Authentication
- Role-based Authorization (ADMIN, ADJUSTER)
- Claim Creation, Validation, and Approval
- PostgreSQL persistence
- Clean layered architecture

## Roles
- **ADJUSTER**: Create and validate claims
- **ADMIN**: Approve claims
